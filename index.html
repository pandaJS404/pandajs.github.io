图片的懒加载与预加载

2020-04-16

懒加载

原理：

先将img标签的src链接设为同一张图片（比如空白图片），然后给img标签设置自定义属性（比如 data-src）,然后将真正的图片地址存储在data-src中，当JS监听到该图片元素进入可视窗口时，将自定义属性中的地址存储到src属性中，并删除data-src属性。达到懒加载的效果。

好处：

这样做能防止页面一次性向服务器发送大量请求，导致服务器响应面，页面卡顿崩溃等

数组

2020-04-16

数组

所有数组都继承于Array.prototype

通过系统内置的Array构造函数申明数组

数组的方法

push方法 想数组的最后一位添加值，返回值是新数组的长度

pop 方法是删除数组的最后一位，返回值是被删除的值

unshift 方法是向数组的第一位添加值，添加到数组的最前面，返回值是新数组的长度

shift 方法是删除数组的第一位，返回值是被删除的值

reverse 方法方法是反转数组，返回值是反转后的数组

splice 方法具有三个参数

第一个参数是要删除的位置数组下标 （负数从后面开始数）

第二个是删除的个数（可以为0，为0则不删除）

第三个是在删除的位置添加新的元素进去（可以多个）

返回值是被删除值的数组

sort 方法对数组进行排序 arr.sort()

括号中填入一个函数

concat 方法连接两个数组，返回值是连接后的新数组

toString 方法是将数组转换成字符串，返回值是转换后的字符串

join 方法是将数组转换为字符串，返回值是分割后的新字符串

参数是规定用什么分割字符串，不写默认以 , （逗号）分割

spit 方法将字符串转换成数组，返回值是转换后的新数组

第一个参数是表示以什么为分割转化为数组(默认是将所有的作为一项)

第二个参数表示截取数组的长度，可不填

indexOf 查找某个元素在数组重的位置 找到返回该元素的索引，没有则返回 -1

Array.from() 方法是将类数组转换成数组

参数是需要转换的类数组

返回值是转换后的新数组

```bash
let str = “foo”;
let a = Array.from(str);

let str = {
0: “a”,
1: “b”,
2: “c”,
length: 3,
slice: Array.prototype.slice
}
console.log(Array.from(str));
``

对于前端学习，必不可少的就是浏览器，基本都是和浏览器打交道

五大主流浏览器，也就是自己拥有内核的浏览器

最老牌的IE 内核为 trident

其次是现目前最火的浏览器chrome 内核为 blink / webkit

然后是 safari 内核为 webkit

再就是 firefox 内核为 gecko

最后就是 opera 内核为 presto

ECMA

ECMA–262规范 脚本语言的规范 ECMAScript

JS是单线程编程语言

它可以通过轮转时间片来模拟多线程，在极短的时间内不停的切换任务

ECMA命名规范：

不能以数字开头，以字母开头，不能使用关键字、保留字

使用语义化，结构化，使用小驼峰命名法

ECMA的数据类型

JavaScript的数据类型分为两种

基本数据类型 也称为原始值

Number String Boolean Undefined Null （ES6新增了 symbol）

引用数据类型 也称为引用值

Object array function date RegExp

运算符

在JavaScript中 + 号不仅仅是加的意思 还可以是字符串拼接

任何数据类型的 + 字符串结果都是字符串

/ 表示除法

0 / 0 会得到 NaN（Not a Number）

NaN与包括自己在内的任何东西都不相等

1 / 0 -> infinity -> 正无穷

-1 / 0 -infinity -> 负无穷

比较运算符

包括 > < >= <= = == === != !==

如果是字符串就比较 ASCII 值

比较运算符

逻辑运算符一定为的值

undefined null NaN “” 0 false 为假，其余的都为真

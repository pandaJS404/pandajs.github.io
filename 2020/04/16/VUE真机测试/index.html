Vue真机测试

修改配置文件 package.json 中的 dev 选项

在 dev 增加中增加 –host 0.0.0.0

或者修改 config 中 index.js host

将localhost改为 0.0.0.0 或者本机的 IP 地址均可

获取本机IP的方式

打开cmd命令 输入 ipconfig 其中的 ipv4 就是你的ip

注：在百度搜索显示的IP不能使用 / 手机和电脑还需要处于同一局域网中
